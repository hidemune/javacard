10. 01.基本　演算やデータの比較　　
サンプルコード

同じであることを確認

If A = B Then
    MsgBox "A=B"
Else
    MsgBox "A<>B"
End If

異なることを確認
If A <> B Then
    MsgBox "A<>B"
End If

AはB以上
If A >= B Then
    MsgBox "A>=B"
End If

AはBより大きい
If A > B Then
    MsgBox "A>B"
End If

AはB以下
If A <= B Then
    MsgBox "A<=B"
End If

AはB+C以上
If A >= B + C Then
    MsgBox "A>=B+C"
End If

複雑になる場合は括弧をつけましょう。
括弧は２重、３重の入れ子にできます。
If (A >= (B + C)) And (B > C) Then
    MsgBox "(A >= (B + C)) And (B > C)"
End If
